# 🫔 La Tamalería Cancún - Página Web

Bienvenidos al repositorio de **La Tamalería Cancún**, una página web informativa diseñada para presentar la historia, el menú y la información de contacto del negocio.  

## 🚀 Características

✅ Diseño responsivo y fácil de navegar.  
✅ Menú de navegación con enlaces a las secciones principales.  
✅ Sección de contacto con formulario funcional.  
✅ Código optimizado y accesible.  
✅ Eliminación del carrito de compras (no necesario por ahora).
